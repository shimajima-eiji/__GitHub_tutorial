# このページは、`pages`ディレクトリのREADME.mdが表示される
ルートディレクトリでは、`index.html`と`README.md`が存在する。
この場合、index.htmlが優先して表示される。

同様に、GitHub Pagesの仕様として、`index.md`と`README.md`も同時に置くことができる。
この場合はどちらが表示されるか、リンクから検証してみよう。
アドレスに注目してみて欲しい
